# Security Policy

## Supported Versions

Currently, we only support the latest version of The Serpent Kingdom for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do NOT disclose it publicly in an issue or discussion. 

Please send an e-mail to the maintainers or report it privately through GitHub's security advisory feature. We will review all reports and respond as quickly as possible. We appreciate your help in keeping The Serpent Kingdom secure!
